ORL Dataset (AT&T)
400 images · 40 subjects · 10 images per subject
Standard benchmark dataset for face recognition. All images taken against dark background with subjects in upright frontal position. Variations in lighting, facial expressions and facial details.
PGM Format 92×112 px Grayscale
FEI / Sheffield Dataset
3780 images · Sheffield Face DB
Sheffield (formerly UMIST) face database. Multiview face images taken in a variety of poses. Used for validation and cross-dataset testing.
JPG Format Multi-view Varied Lighting

SIFT — Scale Invariant Feature Transform
Detects keypoints that are invariant to scale and rotation. Produces 128-dimensional descriptor vectors per keypoint. Robust to illumination changes and partial occlusion.
Output Dimensions128-D
Num Features500
Sigma1.6
HOG — Histogram of Oriented Gradients
Captures edge and gradient structure. Divides image into cells and computes gradient orientation histograms. Excellent for capturing facial shape information.
Output Dimensions~8100-D
Orientations9
Cell Size8×8 px
Gabor — Multi-Frequency Filter Bank
Applies Gabor filters at multiple frequencies and orientations. Captures texture information at different scales. Well-suited for facial texture analysis.
Output Dimensions24-D
Frequencies0.1, 0.3, 0.5
Orientations0°, 45°, 90°, 135°
Canny — Edge Detection
Extracts edge information using Canny edge detector. Provides spatial edge density histogram. Captures structural boundaries of facial features.
Output Dimensions65-D
Low Threshold50
High Threshold150
Custom CNN
4-block convolutional network fused with feature MLP branch. Image branch: Conv32 → Conv64 → Conv128 → Conv256. Feature branch: Dense1024 → 512 → 256. Fusion with residual connection → Dense512 → 256 → 128 → 64 embedding → Softmax output.
Input Shape128×128×1
Feature Dimension8317-D
Embedding Dimension64-D
StatusReady
MobileNetV2 (Transfer Learning)
MobileNetV2 pretrained on ImageNet fine-tuned on ORL/Sheffield datasets. Dual-branch: MobileNetV2 image branch + feature MLP branch. Lightweight and efficient for deployment.
Input Shape96×96×3
Fine-tune FromLayer 100
Embedding Dimension64-D
